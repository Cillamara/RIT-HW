A literal expression is of the prefix form, where {value} is an integer:
 
    '{value}'
 
For example:
 
    '10'
    '4'
 
When emitted, the expressions are displayed infix as strings:
 
    '10'
    '4'
 
When evaluated, their integer form is returns:
 
    10
    4
